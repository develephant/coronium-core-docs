    Setup/Access

    Screencast Available

    Watch the Coronium Core Webmin setup instructions in a screencast format by Clicking here.

    Pages Configuration

    The Coronium Core Webmin is built using the Pages module, you must configure the Pages module first, if you have not already done so.

    Webmin Access

    Google Chrome

    For the best Webmin experience, please use the Google Chrome web browser.

    Important Note

    Be sure to add the trailing slash ( / ) to the webmin address.

    To access the Coronium Core Webmin, navigate to the following address using your browser:

    http://<your-coronium-domain>/_coronium/webmin/

    If you've secured your system using SSL (see HTTPS/SSL), then:

    https://<your-coronium-domain>/_coronium/webmin/

    You will be asked for the Webmin access credentials, the user is cloudwebmin. The default password is cloudadmin.

    To change the webmin password see the Password section of the Webmin Guide.
